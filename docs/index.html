inFactory@ /Studio by XED.dev

AI-Agent-First Ghost Theme Factory — from preset to production in seconds.

Step 1 — Install as root

# As root (installs CLI globally to /opt/infactory):
curl -fsSL https://studio.xed.dev/install.sh | bash

Requires: Node.js 18+ · Git · Ghost CMS 6.x

Step 2 — Setup as Ghost user

# As your Ghost user (e.g. g-host):
cd /var/ghost/your-site
infactory install

# That's it. Your AI agent can now control Ghost via API.
infactory status

All infactory commands run as the Ghost user, never as root.

What it does

inFactory turns YAML preset files into production-ready Ghost themes. Define your design tokens (colors, fonts, spacing), pick sections from the library, and deploy — all from the terminal. Built for AI coding agents, not mouse clicks.

Two components

ComponentPurpose
infactory-cliBuild Ghost themes from YAML presets. Scaffold, preview, deploy.
infactory-serverFactory Floor Controller on your Ghost host. REST API for remote agents.

Server management

CommandWhat it does
infactory installSetup as Ghost companion (auto-detects config)
infactory startStart inFactory server
infactory stopStop inFactory server
infactory restartRestart inFactory server
infactory statusShow Ghost + inFactory status
infactory updateUpdate to latest version
infactory ghost restartRestart Ghost CMS

Auto-sleep after 6 hours of inactivity. Wakes up on next request.

Server API

EndpointWhat it does
POST /api/theme/buildBuild preset into Ghost theme ZIP
POST /api/theme/deployBuild + upload + activate on Ghost
POST /api/ghost/pagesCreate or update a Ghost page
POST /api/ghost/images/migrateMigrate all images of a page to Ghost
